# 介绍

在使用久了vue-cli2之后，由于webpack4的发布，我便尝试的去使用了vue-cli3，但由于一些原因导致我没法使用脚手架进行项目的搭建，所以我便尝试将vue-cli2的配置升级到webpack4中，虽然途中十分坎坷，但这不就是学习新东西的乐趣吗？借此机会，我将我的个人学习历程记录下来，同时也可以用来回顾只是，毕竟温故而知新~
